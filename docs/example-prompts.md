# Ejemplos de prompts

## Ejemplo 1 — Debate estratégico

```text
ultracode: usa Debate Council para evaluar esta tesis:

Tesis: La empresa debería crear un sistema interno de agentes de IA para validar compras antes de aprobarlas.

Contexto: Hay órdenes de compra pendientes, histórico SAP, cotizaciones PDF y riesgo de sobreprecio.

Objetivo: decidir si vale la pena desarrollar un piloto.
```

## Ejemplo 2 — Debate de inversión

```text
ultracode: usa Debate Council para evaluar esta tesis:

Tesis: Tiene sentido mantener una posición en una acción tecnológica altamente volátil hasta el vencimiento de opciones del mes.

Contexto: Tengo acciones y opciones; necesito evaluar riesgo, escenarios y alternativas.

Objetivo: obtener argumentos a favor, en contra, riesgos, sesgos y veredicto no personalizado.
```

## Ejemplo 3 — Debate de proceso

```text
/debate-council Tesis: El proceso de transporte inbound debe pasar de Excel y WhatsApp a un flujo SAP/BTP con trazabilidad documental completa. Contexto: OC tardía, guías no vinculadas y HES incompleta. Objetivo: evaluar la tesis antes de presentar propuesta.
```
